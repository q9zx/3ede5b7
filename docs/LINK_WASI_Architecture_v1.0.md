# LINK_WASI_Architecture_v1.0

## Objetivo

Framework de publicación inmobiliaria para Metri.immo.

## Flujo

1.  Ejecutar LINK WASI indicando repositorio y carpeta.
2.  Leer DESCRIPCION.docx como fuente única.
3.  Detectar imágenes automáticamente.
4.  Seleccionar Hero y Open Graph.
5.  Generar `index.html`, `styles.css` y `script.js`.
6.  Ejecutar Quality Report.

## Reglas

-   Mobile Premium Shell.
-   Usar únicamente `favicon.png`.
-   Sin sección de contacto visible.
-   Sección **Diferenciales**.
-   Texto justificado.
-   SEO completo: Open Graph, Twitter Cards, JSON-LD.
-   Responsive first.
-   Carrusel ordenado por impacto comercial.
