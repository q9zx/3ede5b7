# Link Wasi Framework

Sitio inmobiliario estático para GitHub Pages, Netlify o Vercel.

## Publicación con GitHub Pages

1. Sube el contenido de la carpeta `abk` al repositorio `q9zx/3ede5b7`.
2. En GitHub, abre **Settings → Pages**.
3. Elige la rama `main` y la carpeta `/ (root)`.
4. El sitio estará disponible en `https://q9zx.github.io/3ede5b7/abk/`.

No requiere proceso de instalación ni dependencias.
